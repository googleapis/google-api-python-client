search(resource, pageSize=None, pageToken=None, query=None, x__xgafv=None) 
Search Product resources that consumers have access to, within the
scope of the consumer cloud resource hierarchy context.

Args:
  resource: string, Required. The name of the resource context. See
SearchCatalogsRequest.resource for details. (required)
  pageSize: integer, The maximum number of entries that are requested.
  pageToken: string, A pagination token returned from a previous call to SearchProducts that
indicates where this listing should continue from.
This field is optional.
  query: string, The query to filter the products.

The supported queries are:
* List products of all catalogs: empty
* List products under a catalog: `parent=catalogs/{catalog_id}`
* Get a product by name:
`name=catalogs/{catalog_id}/products/{product_id}`
  x__xgafv: string, V1 error format.
    Allowed values
      1 - v1 error format
      2 - v2 error format
